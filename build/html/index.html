Triangle Module¶

class triangle.Triangle(*args)¶

Bases: object

Represents a triangle with properties for its base, right side, left side, and height. Provides functionality to calculate the perimeter and area of the triangle.

Variables:

  • _base – The base of the triangle. Default value is 1.

  • _right_side – The right side of the triangle. Default value is 1.

  • _left_side – The left side of the triangle. Default value is 1.

  • _height – The height of the triangle. Default value is 1.

  • was_defaulted – True when the input values for a constructor are incorrect. Default is False.

area()¶

Calculates and returns the area of the triangle.

Returns:

The area of the triangle calculated using the base and height.

property base¶

The base of the triangle. Must be a positive number greater than zero.

get_triangle_type()¶

Determines the type of the triangle based on its sides.

Returns:

A string indicating whether the triangle is Equilateral, Isosceles, or Scalene.

property height¶

The height of the triangle. Must be a positive number greater than zero.

property left_side¶

The left side of the triangle. Must be a positive number greater than zero.

perimeter()¶

Calculates and returns the perimeter of the triangle.

Returns:

The sum of all sides of the triangle.

property right_side¶

The right side of the triangle. Must be a positive number greater than zero.
